# 📚 StudyMateAI

StudyMateAI is an AI-powered learning assistant built with Streamlit and Ollama.

## Features

- 📄 Upload PDF study notes
- 📝 Generate AI-powered summaries
- 🎓 Ask questions from your notes
- ❓ Generate quizzes automatically
- 📊 View learning analytics
- 🧠 Track learning history

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3
- Pandas

## How It Works

1. Login to the application
2. Upload a PDF
3. Extract text from the PDF
4. Generate summaries
5. Chat with the AI tutor
6. Create quizzes automatically

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
